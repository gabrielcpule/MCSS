# MCSS Login Page (TypeScript)

This project demonstrates a login page built with TypeScript and styled using the MCSS framework. The design is based on the provided image reference and follows MCSS component guidelines.

## Features
- Responsive login form
- Email and password fields
- Password visibility toggle
- Social login buttons (Google, LinkedIn, Facebook, GitHub)
- MCSS-based styling

## Getting Started

1. Install dependencies:
   ```
npm install
   ```
2. Build the TypeScript code:
   ```
npm run build
   ```
3. Start the development server:
   ```
npm start
   ```
4. Open `http://localhost:3000` to view the login page.

## File Structure
- `index.html` – Main login page
- `src/login.ts` – TypeScript logic for the login form
- `dist/login.js` – Compiled JavaScript
- `../mcss.css` – MCSS framework (imported)

## MCSS Reference
See `../mcss.css` and `../documentation.md` for MCSS component usage and documentation.
